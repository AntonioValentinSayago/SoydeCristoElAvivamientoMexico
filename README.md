# SoydeCristoElAvivamientoMexico
Aplicación web desarrollada para gestionar el registro de asistentes al evento Soy de Cristo México 2026 de las Iglesias Ebenezer. La plataforma permitirá que cada persona pueda registrarse al evento de forma sencilla y, dentro del mismo flujo, también pueda realizar la compra de productos oficiales, como playeras.
